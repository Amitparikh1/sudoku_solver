# sudoku_solver - created using TechByTim tutorial
A sudoku solver in python using a backtracking algorithm and a GUI 
INSTRUCTIONS FOR GUI:
- click on a space to select it 
- enter number after clicking using keyboard
- press 'return' key to submit a number on the board, use 'delete' to remove it
- if an invalid number is entered it won't be displayed
- press the spacebar to show the solved puzzle and watch the backtracking algorithm at work
